# System Info Tool v6.0

Универсальная программа с графическим интерфейсом для получения полной информации о вашем компьютере. Работает на Windows, Linux и macOS. Не требует установки — достаточно запустить файл.


## Возможности
- **Общая информация:** ОС, имя компьютера, пользователь, время работы, производитель, модель, серийный номер.
- **Процессор:** модель, ядра, потоки, частота, загрузка, температура (при поддержке).
- **Оперативная память:** использование ОЗУ и Swap.
- **Видеокарта:** модели, драйвер, объём памяти.
- **Диски:** устройство, модель, размер, тип (SSD/HDD), занятое и свободное место.
- **Сеть:** интерфейсы, IP-адреса, MAC-адреса, статус, трафик.
- **Активные сетевые соединения:** протокол, локальный и удалённый адреса, PID процесса.
- **Батарея:** уровень заряда, время работы, источник питания.
- **Материнская плата и BIOS:** производитель, модель, версия, серийный номер.
- **Аудиоустройства:** список.
- **USB-устройства:** идентификаторы и описание.
- **Сенсоры:** температуры, вентиляторы (если доступны).
- **Службы/демоны:** список и статус.
- **Установленное ПО:** из реестра Windows, dpkg/rpm (Linux) или Homebrew (macOS).
- **Процессы:** топ-50 по загрузке CPU.
- **Мониторинг в реальном времени:** графики загрузки CPU и RAM.
- **Экспорт отчёта:** TXT, JSON, CSV, HTML.
- **Копирование в буфер обмена:** весь отчёт одной кнопкой.
- **Настройки:** светлая/тёмная тема, интервал автообновления.
- **Кроссплатформенность:** единый код для Windows, Linux и macOS.

## Требования
- Python 3.6 или выше (для запуска из исходников).
- Рекомендуется установить модуль `psutil` для расширенной информации:
- - На Windows может использоваться WMI или PowerShell (доступны по умолчанию).

## Установка и запуск
### Из исходного кода
1. Скачайте файл `system_info.py` из этого репозитория.
2. Установите зависимости (опционально): `pip install psutil`
3. Запустите: `python system_info.py`

### Готовые исполняемые файлы
В разделе [Releases](https://github.com/yourname/system-info-tool/releases) вы можете скачать скомпилированные версии для Windows (.exe) и, возможно, для Linux/macOS. Просто запустите файл — никакой установки не требуется.

## Безопасность и конфиденциальность
- Программа работает **полностью локально** и **не передаёт никакие данные** по сети.
- Все собираемые сведения отображаются только на вашем экране или сохраняются в файл по вашему желанию.
- Вы сами контролируете, кому показывать или отправлять отчёт.
- Исходный код открыт — вы можете проверить его самостоятельно.
