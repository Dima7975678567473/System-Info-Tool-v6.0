# System-Info-Tool-v6.0
Кроссплатформенное приложение с графическим интерфейсом для просмотра детальной информации о компьютере: процессор, оперативная память, видеокарта, диски, сеть, батарея, материнская плата, USB, сенсоры, процессы и многое другое. Работает на Windows, Linux и macOS. Все данные собираются локально и никуда не передаются.
